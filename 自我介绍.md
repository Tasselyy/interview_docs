**UTMIST 团队担任 AI 研发工程师。** 我主要面向学术研究场景，从零到一设计并开发了一套解耦式的 Agentic RAG 系统。在 Agent 端，我用 LangGraph 构建了基于意图驱动的多路由架构，并亲自基于 Qwen2.5 做了 LoRA 微调和 DPO 对齐训练，专门优化了模型的路由选择和判答逻辑，显著降低了低质量回答的误放行率和 API 成本。在 RAG 端，我落地了 Dense 加 BM25 的双路检索，并且通过 MCP 协议实现了检索服务与 Agent 的标准化解耦，保证了系统的高可扩展性。最后这套模型我经过 INT4 量化后使用了 vLLM 进行部署。

**一个金融 Multi-Agent 深度研究助手项目。** 针对金融场景任务复杂、需要高度可溯源的特点，我设计了一套中心化的多智能体编排架构。我把像财报分析、新闻扫描这些标准的分析流程抽象成了底层的 Skills 单元，让主 Agent 能够并行派发任务给不同的子 Agent。最后我还构建了结论和证据的对齐机制，确保生成的每一份研究报告都是结构化且完全可追溯的。
